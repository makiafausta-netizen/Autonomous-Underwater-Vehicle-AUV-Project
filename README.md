# Autonomous Underwater Vehicle (AUV) Project

**Role:** Software Developer / Embedded Systems Engineer  
**Technologies:** Python, C++, ROS2, Linux, Arduino/STM32, IMU, Ultrasonic, Pressure Sensors, Git  
**Focus Areas:** Autonomous Navigation, Sensor Fusion, Real-Time Control, Embedded Software, AI & Robotics

## Project Overview
Developed modular, real-time software for an Autonomous Underwater Vehicle (AUV) enabling autonomous navigation, obstacle avoidance, and mission execution in dynamic underwater environments. The project emphasizes software architecture, sensor data processing, low-latency decision-making, and system reliability—mirroring challenges in real-world robotics deployments.

## Key Contributions
- Implemented ROS2-based software modules for navigation, control, and mission logic using Python and C++.  
- Built **sensor fusion pipelines** combining IMU, ultrasonic, and pressure data for accurate **state estimation** and robust autonomous behavior.  
- Developed **event-driven, real-time logic** for obstacle detection, mission sequencing, and system state management.  
- Conducted extensive **simulation and live-environment testing**, debugging runtime issues, sensor inconsistencies, and timing challenges.  
- Refined control algorithms and error handling to improve **reliability and deterministic behavior** under resource-constrained conditions.  
- Authored technical documentation covering software architecture, data flows, and testing protocols to support maintainability and collaboration.

## Outcome & Achievements
- Successfully deployed autonomous navigation and obstacle avoidance software in real-world underwater tests.  
- Strengthened expertise in **embedded systems, real-time software, sensor processing, and autonomous navigation**—directly relevant to robotics and AI-driven systems.  
- Demonstrated practical experience in **end-to-end robotic autonomy** applicable to control, navigation, and AI model deployment.
